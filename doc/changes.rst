Release Notes

0.3.1 (unreleased)
------------------

* Add ``--version`` option.
* Add Python 3.6, remove 3.3.

* Add tests to util subpackage.
* Add ``__future__`` statements.
* Clean up API documentation.
* Minor bug fixes.

0.1.0 (2015-03-25)
------------------

* Initial release.  Used to scan all SDSS data.
